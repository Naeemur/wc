Hello Internet

I am Md. Naeemur Rahman. I am a full-stack web developer and tech enthusiast. I build scalable cloud applications with JavaScript technologies like Node JS, jQuery, Electron, React JS and more. I am an expert in building cloud infrastructures on IaaS and PaaS services. I also experiment with new technologies  and I prefer to build my own libraries, modules and frameworks to use on my projects. I usually work on projects remotely from home on contract basis, but I like to work in a good office environment. I have been working full time since 2018.
